# Spring Boot Recipe Application
## This is a test application. 
## With its help I develop my skyls on the backend side about Java, Spring MVC, Spring JPA, Spring Boot, Maven, Hibernate, SQL, databases, Thymeleaf. 
## The front part we accomplished it with the help of the Bootstrap library that always offers me a solution when I need HTML and CSS organized. 
## The application is in version not completed . 
## At this point, the application contains the main and necessary modules for a web application as well as the test modules. 
## In the future, the application will also contain authentication modules.
